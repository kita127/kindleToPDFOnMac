# kindleToPngOnMac

## 使用ツール

* Kindle


## 覚書

* ウィンドウサイズ計測
    * Command + Shift + 4
    * はじめに top と left が表示される
    * 左クリックを押し矩形選択をすると選択した箇所からの width と height が表示される
* PDF への変換
    * finder で PDF にした png ファイルを全て選択する
    * 右クリック → クイックアクション → PDF を作成

## 注意

* 実行中は他操作ができません
* 出力先フォルダが既に存在する場合は実行できません
